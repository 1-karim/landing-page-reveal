# REVEAL LANDING PAGE EXP 16

A small interactive demo showcasing a simple reveal animation for a portfolio website
Features
- Timeline animations built with GSAP
- Supports sequential image assets from the public folder

# Preview

![Preview](/screenshot.gif)

# Quick start
1. Install dependencies: npm install
2. Start the dev server: npm run dev
3. Place image assets in public/ named as slide-img-[n].jpg (e.g. slide-img-1.jpg, slide-img-2.jpg)
4. Open the local URL shown in the terminal

# Project structure
- index.html — markup
- styles.css — layout and styles
- script.js — initialization, asset preloading, and animation logic
- public/ — static assets (images, gif preview)

# Notes
- Ensure image filenames are sequential and match the expected pattern used by the script.
- GSAP is used for timeline and interaction smoothing.
